HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Made up of elements (text elements, image elements, header elements, list elements) (meta-related elements, content division elements, semantic elements)
  • HTML works through the use of tags
  • html lang -- sets webpage language. !DOCTYPE -- sets document type (html). head -- contains metadata. title -- title. style -- defines the associated CSS styles. base -- the base URL. link -- connects an external source like a CSS or JavaScript file. mets -- defines the metasata (character set, description, keywords, author, and viewpoint). script -- embeds client side scripts in the HTML document
  • Common body tags
  • h1, h2, h3 ... -- headers. p -- paragraph text. ul, ol, li -- unordered lists, ordered lists, list items. img -- image info. br -- creates a section/page break. a -- "anchor" creates links to webpages

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Applies styles to HMTL tags by using the tag as the "selector"
  • selector {property: value}
  • EX: h1 {color: blue}

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • How to open new branch:
  1. git status
  2. git checkout -b feature/branchName
  • Commit to branch steps:
  1. git status (make sure youre on the right branch)
  2. git add -A
  3. git pull origin main
  4. git push origin feature/branchName
  5. review and merge in GitHub to commit

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional if statements
  • For loops
  • Functions must be called to appear
